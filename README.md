# Interactive Periodic Table

Welcome to the Interactive Periodic Table project! This web application provides a dynamic and responsive periodic table of elements, designed using HTML, CSS, JavaScript, and Tailwind CSS. The table offers detailed information on each element and adapts to different screen sizes for a seamless user experience.

## Features

- **Responsive Design:** The periodic table resizes for various screen sizes, from medium devices to mobile devices.
- **Horizontal Scrolling on Mobile:** On mobile devices, the table is placed in a subcontainer with horizontal scrolling for easy navigation.
- **Element Details Modal:** A long click on any element opens a closable modal box displaying detailed information about the element.

## Data Format

The periodic table displays the following information for each element:

- `atomicNumber`: Atomic number of the element.
- `symbol`: Chemical symbol of the element.
- `name`: Name of the element.
- `atomicMass`: Atomic mass of the element.
- `electronicConfiguration`: Electronic configuration of the element.
- `electronegativity`: Electronegativity of the element.
- `atomicRadius`: Atomic radius of the element.
- `ionRadius`: Ionic radius of the element.
- `vanDerWaalsRadius`: Van der Waals radius of the element.
- `ionizationEnergy`: Ionization energy of the element.
- `electronAffinity`: Electron affinity of the element.
- `oxidationStates`: Oxidation states of the element.
- `standardState`: Standard state of the element (solid, liquid, gas).
- `bondingType`: Bonding type of the element.
- `meltingPoint`: Melting point of the element.
- `boilingPoint`: Boiling point of the element.
- `density`: Density of the element.
- `groupBlock`: Group block of the element.
- `yearDiscovered`: Year the element was discovered.
- `block`: Periodic block of the element.
- `cpkHexColor`: CPK color code of the element.
- `period`: Period of the element in the periodic table.
- `group`: Group of the element in the periodic table.
